# NeverDie Launcher

Лаунчер для Minecraft 1.16.5 (Vanilla / Forge / Fabric / OptiFine) на PyQt6.

---

## 1. Что нужно установить перед запуском

### 1.1 Python
Нужен **Python 3.10+** (рекомендую 3.11). Скачать: https://www.python.org/downloads/
При установке на Windows обязательно поставь галочку **"Add Python to PATH"**.

### 1.2 Java
Сам Minecraft 1.16.5 запускается через Java. Нужна **Java 8** (для совместимости со старыми модами/OptiFine) или **Java 16/17**.
Скачать: https://adoptium.net/ (выбери версию 8 или 17 для Windows x64, .msi инсталлятор).

Проверка после установки — открой `cmd` и напиши:
```
java -version
```
Должна вывестись версия Java. Если пишет "не является внутренней или внешней командой" — Java не добавлена в PATH, переустанови с галочкой "Add to PATH" или укажи путь к `javaw.exe` вручную в `config.json` (поле `java_path`).

### 1.3 Python-библиотеки
В папке лаунчера открой `cmd` / PowerShell и выполни:
```
pip install -r requirements.txt
```
Это установит **PyQt6** — единственную внешнюю библиотеку, которая используется (для интерфейса).

## 2. Структура папок

```
neverdie_launcher/
├── main.py                  ← главный файл, запускать его
├── mc_downloader.py         ← скачивание vanilla Minecraft
├── mc_launcher.py           ← сборка classpath и запуск игры
├── modloader_installer.py   ← установка Forge/Fabric/OptiFine
├── requirements.txt
├── config.json              ← создаётся автоматически (ник, ОЗУ, выбранный загрузчик)
└── assets/
    ├── background.png       ← фон лаунчера (уже готов, не трогай)
    └── logo.png             ← СЮДА положи свой логотип (см. ниже)
```

## 3. Куда класть логотип

Положи файл с названием **`logo.png`** в папку:
```
neverdie_launcher/assets/logo.png
```
Рекомендуемый размер — квадратный, минимум 128×128 px (PNG с прозрачностью будет смотреться лучше всего). Лаунчер сам подхватит его и отобразит в левом верхнем углу при следующем запуске. Если файла нет — там будет временная плашка "LOGO", чтобы не сломать вёрстку.

## 4. Запуск лаунчера

```
python main.py
```

## 5. Как это работает (важные тонкости)

### 5.1 Vanilla 1.16.5
При первом запуске (если выбран режим "Vanilla") лаунчер САМ скачивает:
- `client.jar` версии 1.16.5,
- все нужные библиотеки,
- ассеты (звуки, языковые файлы и т.п.)

прямо с официальных серверов Mojang (`piston-meta.mojang.com`, `resources.download.minecraft.net`). Никаких сторонних серверов. Загрузка идёт в папку:
```
%APPDATA%\.neverdie_minecraft\
```
Это и есть аналог `.minecraft`. Первая загрузка может занять от 1 до 5 минут в зависимости от интернета (ассетов там много, по умолчанию используется многопоточная загрузка — 8-12 потоков).

### 5.2 Fabric — устанавливается полностью автоматически
Лаунчер обращается к официальному **Fabric Meta API** (`meta.fabricmc.net`), скачивает профиль и нужные библиотеки. Никаких действий от тебя не требуется — просто выбери "Fabric" в выпадающем списке и нажми "Запустить".

### 5.3 Forge — устанавливается автоматически, но через официальный installer.jar
В отличие от Fabric, у Forge нет простого API "дай мне готовый профиль json". Официальный (и единственный поддерживаемый Forge-проектом) способ — скачать `forge-installer.jar` и запустить его с флагом `--installClient`. Лаунчер делает это сам:
1. Скачивает `forge-<версия>-installer.jar` с `maven.minecraftforge.net`.
2. Запускает `java -jar forge-installer.jar --installClient <папка>`.
3. Forge сам создаёт нужную папку версии внутри `versions/`.

Из-за того, что Forge installer — это реальная Java-программа, на первую установку может уйти 10-40 секунд (видно по прогресс-бару "Установка Forge...").

### 5.4 OptiFine — ОДНО ручное действие (не наша вина, а защита самого OptiFine)
OptiFine **не отдаёт прямых ссылок на скачивание** — сайт `optifine.net` заставляет проходить через рекламную страницу и кнопку "Download" с защитой от ботов. Автоматизировать это было бы нарушением их защиты и сломалось бы при любом обновлении сайта.

Поэтому для OptiFine процесс такой:
1. Зайди на https://optifine.net/downloads , выбери версию для **1.16.5**, скачай `OptiFine_1.16.5_HD_U_xxx.jar` (через их жёлтую кнопку Download, подождав таймер).
2. В лаунчере выбери "OptiFine" в выпадающем списке — появится кнопка **"Выбрать OptiFine_installer.jar..."**.
3. Укажи скачанный файл.
4. Нажми "Запустить" — откроется отдельное окошко самого OptiFine с кнопкой **Install**. Нажми её один раз.
5. После этого нажми "Запустить" в лаунчере ещё раз — теперь версия с OptiFine уже будет найдена автоматически и игра запустится.

Это нужно сделать только один раз на каждую новую версию OptiFine.

### 5.5 Никнейм / вход
Используется **offline-режим** (как в пиратках) — никнейм просто вводится в текстовое поле на вкладке "Запуск" и сохраняется локально в `config.json`. Никакой авторизации через Microsoft не требуется. Это значит, что игра запустится без официальной лицензии, но на сервера со включённым `online-mode` (официальная проверка лицензии) зайти не получится — только на сервера с `online-mode=false` или в одиночную игру.

### 5.6 ОЗУ (RAM)
На вкладке "Настройки" ползунок выставляет `-Xmx` (максимум памяти для Java) от 1024 до 16384 МБ. Значение сохраняется в `config.json` и применяется при следующем запуске.

### 5.7 Кнопка "Открыть папку версии"
Открывает `%APPDATA%\.neverdie_minecraft` в Проводнике Windows — там лежат все версии, моды (для Forge/Fabric нужно будет руками докинуть `.jar` моды в подпапку `mods/`, которую Forge/Fabric создаёт сами при первом запуске), скины, сохранения и т.д.

## 6. Известные ограничения (важно знать)

- **Только Windows** — пути и `os.startfile()` рассчитаны на Windows. На Linux/Mac кнопка "Открыть папку" просто покажет путь текстом вместо открытия проводника.
- **Offline-никнейм** — не настоящая лицензия Mojang/Microsoft. Скины не подгружаются с серверов Mojang (будет дефолтный Steve/Alex), мультиплеер ограничен серверами с `online-mode=false`.
- **OptiFine требует одно ручное нажатие** — см. пункт 5.4, технически не обойти без нарушения защиты optifine.net.
- **Forge/Fabric моды** — лаунчер ставит только сам загрузчик. Сами файлы модов (`.jar`) нужно класть в папку `mods/` вручную (она появится в `.neverdie_minecraft/mods/` после первого запуска с Forge или Fabric).
- Окно лаунчера сделано без рамки (`frameless`) — для перетаскивания зажми и тащи за любую свободную область фона, для закрытия/минимизации — крестик/минус в правом верхнем углу.

## 7. Цветовая схема (если захочешь поменять)

Все цвета вынесены в константы в начале `main.py`:
```python
RED = "#E5262B"
RED_BRIGHT = "#FF3B41"
RED_DARK = "#7A0F12"
BLACK = "#0A0606"
BLACK_SOFT = "#140A0A"
WHITE = "#F5F0F0"
```
Меняешь значения тут — меняется вся палитра во всём интерфейсе.
